# orderPorts
## 菜单位置
Port->orderPorts

## 功能描述
对3D Layout的Port按照Net进行排序，排序规则为：
1. 器件
2. 总线(Net上的数字)
3. 差分线P/N, +/-, P/M


## 操作说明
保持3D Layout窗口在最前面，运行Port->orderPorts命令。

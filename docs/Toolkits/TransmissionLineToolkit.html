    Q2D走线建模与优化设计工具 (TransmissionLineToolkit)

    工具概述

    TransmissionLineToolkit 是一个基于 Ansys Q2D Extractor 的传输线建模与仿真工具，提供图形用户界面（GUI）支持多种PCB走线类型的参数化建模、特性阻抗估算和全波仿真。工具通过 IronPython 集成到 AEDT 环境中运行，可直接在 AEDT 内部启动并创建 Q2D 设计。

    系统需求

    • 运行环境: Ansys Electronics Desktop（AEDT）2022 R1 或更高版本
    • 脚本引擎: IronPython（AEDT 内置）
    • 设计类型: Q2D Extractor

    工具位置

    工具位于 Toolbox 安装目录下：

    Toolbox安装目录/
├── UserLib/
│   └── Toolkits/
│       └── q2d_tlineToolkit/
│           ├── TransmissionLineToolkit.py   （主入口脚本）
│           ├── Q2D走线建模与优化设计.pdf   （使用文档）
│           └── TLine/
│               ├── MainForm.py             （GUI主窗体）
│               ├── anstDebug.py            （日志模块）
│               └── Images/                （界面图片资源）
│                   ├── ANSYSlogo.png
│                   ├── WelcomeScreen.png
│                   ├── MS_Single.png
│                   ├── etch1.png
│                   └── ...
└── ...

    在 Toolbox 菜单中调用

    在 Toolbox 菜单 XML 中添加以下配置即可从 Toolbox 直接调用：

    <SubMenu Type="MenuItem"
         Name="TransmissionLine Toolkit"
         ExecuteType="IronPython"
         Path="$UserLib/Toolkits/q2d_tlineToolkit/TransmissionLineToolkit.py"
         EntryFunc="main"
         Arguments="" />

    支持的走线类型

    工具支持以下6种常见PCB传输线结构：

    走线类型 说明 应用场景
    Microstrip - Single Ended 单端微带线 表层单端信号
    Microstrip - Differential 差分微带线 表层差分信号（如USB、HDMI）
    Stripline - Single Ended 单端带状线 内层单端信号
    Stripline - Differential 差分带状线 内层差分信号（如PCIe、DDR）
    Stripline - Broadside Coupled 宽边耦合带状线 高密度差分布线
    Grounded Coplanar Waveguide 接地共面波导（GCPW） 高频微波信号、RF走线

    单位设置

    支持以下4种尺寸单位，全局生效：

    单位 说明
    mil 密耳（推荐，PCB常用）
    inches 英寸
    microns 微米
    millimeters 毫米

    走线参数详解

    基本尺寸参数

    参数名 适用结构 说明
    Trace Width 所有类型 走线宽度
    Trace Thickness 所有类型 走线铜厚度
    Trace Spacing 多走线结构 走线之间的间距
    Number of Traces 单端/差分 走线数量
    Differential Spacing 差分结构 差分对内间距
    Plane Thickness 所有类型 参考地平面铜厚
    Top Dielectric 带状线/GCPW 走线上方介质厚度
    Bottom Dielectric 所有类型 走线下方介质厚度
    Solder Mask Thickness 微带线 阻焊层厚度

    介质材料参数

    每层介质独立设置以下参数：

    参数 默认值 说明
    Er 4.0 相对介电常数
    Tand 0.02 损耗正切
    频率参考 1 GHz 介质参数对应的参考频率
    频率单位 GHz Hz/kHz/MHz/GHz 可选

    支持多层介质配置（带状线）： - Top Dielectric - 走线上方介质（D1） - Bottom Dielectric - 走线下方介质（D2） - D4 Dielectric - 第四层介质（宽边耦合结构） - D5 Dielectric - 第五层介质（宽边耦合结构） - Solder Mask - 阻焊层（微带线，可选）

    高级参数设置

    标签页1：仿真参数

    参数 默认值 说明
    Solution Frequency 10 GHz Q2D求解频率
    Frequency Sweep None 频率扫描类型
    Start/Stop/Step - 扫描范围和步长

    频率扫描类型： - None - 仅单频点求解 - Discrete - 离散频率点扫描 - Interpolating - 插值频率扫描（推荐，速度快）

    标签页2：刻蚀（Etching）

    刻蚀参数模拟实际PCB制造工艺中走线截面的梯形形状：

    设置 说明
    No Etch 不添加刻蚀（矩形截面）
    Over Etch（Top） 顶部过刻蚀，走线顶部比底部窄
    Under Etch 欠刻蚀，走线顶部比底部宽
    Etch Factor 刻蚀百分比（通过滑块调节）

    微带线结构还支持上方（Top Etching）和下方（Bottom Etching）两侧刻蚀方向

    标签页3：表面粗糙度（Surface Roughness）

    支持三种铜箔表面粗糙度模型：

    模型 参数 说明
    No Roughness - 理想光滑表面
    Hammerstad 表面粗糙度 Rq（Top/Sides/Bottom各设置） 经典粗糙度模型
    Huray（Hall-Huray） 粗糙度 + Hall-Huray比值（Top/Sides/Bottom各设置） 高精度物理模型

    分区域设置： - Top - 走线顶面（铜光面） - Sides - 走线侧面 - Bottom - 走线底面（与介质接触面，粗糙度最高）

    粗糙度单位： um / mil / nm 可选

    电镀层（Plating）设置

    参数 说明
    Enable Plating 是否启用电镀层
    Plating Material 电镀材料（默认：copper）
    Plating Thickness 电镀厚度
    Units mil/um/mm 可选

    仿真操作流程

    操作步骤

    1. 选择走线类型（设计类型下拉框）
      ↓
2. 输入走线几何尺寸参数
      ↓
3. 设置介质材料属性（Er、Tand、参考频率）
      ↓
4. （可选）配置刻蚀参数
      ↓
5. （可选）配置表面粗糙度模型
      ↓
6. 配置仿真求解频率和扫描设置
      ↓
7. 选择导出模型格式
      ↓
8. 点击 [Simulate] 按钮
      ↓
Q2D 设计在 AEDT 中自动创建并运行仿真
      ↓
查看仿真结果（Zo、差分阻抗等）

    特性阻抗估算（Calculate）

    点击 Calculate 按钮（如显示）可通过 IPC 经验公式 快速估算特性阻抗 Zo：

    • 无需运行仿真，即时计算
    • 结果显示在界面 Zo = xx Ω 位置
    • 差分结构还显示差分阻抗

    注意：IPC 公式为近似计算，精度低于 Q2D 全波仿真

    仿真结果

    运行仿真后，AEDT 将自动： 1. 创建新的 Q2D Extractor 设计 2. 建立走线几何模型 3. 设置材料属性和边界条件 4. 运行仿真求解 5. 显示特性阻抗 Zo 和其他电参数

    模型导出格式

    仿真完成后，支持将 Q2D 结果导出为电路模型：

    格式 说明 用途
    Tabular W-element RLGC 频变矩阵格式 电路仿真（HSPICE/ADS等）
    Lumped Element 集总参数 RLC 模型 快速预估，低精度

    导出路径设置： - 点击 Model Location 按钮选择导出目录 - 支持直接导入到 Circuit 设计

    GUI 显示控制

    选项 说明
    GUI Show 运行时显示 AEDT GUI 窗口
    GUI Hide 静默运行（加速批量仿真）

    多设计批量建模

    工具支持在同一 AEDT 会话中连续创建多个 Q2D 设计：

    • 每次点击 Simulate 自动递增设计编号（designcounter）
    • 设计名称格式：TLine_1、TLine_2、...
    • 无需重启工具即可切换走线类型继续建模

    常用设计参数参考

    典型微带线参数（FR4，50Ω）

    参数 典型值
    走线宽度 ~90 mil（1oz铜，4mil介质）
    铜厚 1.4 mil（1oz） / 2.8 mil（2oz）
    介质厚度 4 ~ 8 mil
    Er（FR4） 3.9 ~ 4.5（1GHz）
    Tand（FR4） 0.015 ~ 0.025

    典型差分带状线参数（100Ω差分）

    参数 典型值
    走线宽度 ~4 mil
    差分对间距 ~5 mil
    介质厚度（上/下各） 4 ~ 8 mil

    故障排查

    常见问题

    问题 原因 解决方案
    工具无法启动 未在 AEDT 内执行 必须通过 AEDT 的 IronPython 调用
    图片无法加载 Images目录缺失 检查 TLine/Images/ 是否完整
    仿真报错 Q2D 许可证不足 检查 AEDT License 状态
    Zo计算值异常 参数超出有效范围 参照 IPC 规范检查输入参数
    导出模型为空 仿真未完成 等待仿真完成后再导出

    日志查看

    工具使用 anstDebug 模块记录日志： - 默认级别: INFO - 日志位置: AEDT 临时目录（oDesktop.GetTempDirectory()） - 日志文件名: TLine_*.log

    如需详细调试日志，修改 TransmissionLineToolkit.py 中的级别：

    # 将 logging.INFO 改为 logging.DEBUG
oLog = anstDebug.anstDebug('TLine', logging.DEBUG, oDesktop)

    技术说明

    仿真原理

    工具基于 Ansys Q2D Extractor 进行二维横截面电磁场求解：

    1. 建模: 根据输入参数自动创建走线横截面几何体
    2. 材料赋值: 导体、介质材料属性（含频变特性）
    3. 求解: 静电场和磁场方程组求解，提取 RLGC 参数
    4. 后处理: 计算特性阻抗 Zo、差分阻抗 Zdiff 等电气参数

    粗糙度模型说明

    模型 物理意义
    Hammerstad 基于表面均方根粗糙度 Rq 的经验修正因子，适用工程快速评估
    Hall-Huray 基于铜球堆叠模型的物理模型，精度更高，需要两个参数：表面粗糙度和 Hall-Huray 比值

    与 Toolbox 主程序关系

    Toolbox（AedtToolbox.exe）
    ↓ 右键菜单
HFSS3DLayoutDesign.xml / Default.xml
    ↓ IronPython ExecuteType
TransmissionLineToolkit.py
    ↓ 启动 GUI
TLine/MainForm.py (IronPython + Windows Forms)
    ↓ 用户操作
Q2D Extractor 设计（在 AEDT 中创建）
